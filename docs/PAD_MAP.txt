JAMMING TIPS:
- Pads 5–12 are set to LOOP mode — layer them freely
- Pads 1–4 are GATE mode — hold to sustain, release to stop
- All keys are diatonic to C major — everything harmonizes across banks
- BPMs (112/120/128/130) cluster tight — cross-bank mixing works
- Use HOLD pad to keep sounds playing after release
- Layer effects (Filter, Lo-Fi, Isolator) while performing
- Bank A is yours — resample anything into it on-device
