## Adding More Packs

MusicRadar SampleRadar has hundreds of free packs. Browse at:
https://www.musicradar.com/news/tech/free-music-samples-royalty-free-loops-hits-and-multis-702616

To add a new pack:
1. Download the ZIP to `~/Downloads/`
2. Unzip to `_UNZIPPED/` folder
3. Run `organize_library.py` to categorize into the library
4. Update `pick_best_samples.py` bank mappings if needed
5. Re-run the pipeline
