## Bank Concepts Overview

### 1. Big Beat Blowout
**Influences:** Chemical Brothers, Prodigy, Fatboy Slim
**Core Elements:** Breakbeats, acid bass, rave stabs, heavy drums, filtered synths
**BPM Range:** 120-140 BPM
**Key Sounds Needed:** Dirty breaks, 303 acid lines, vocal stabs, distorted percussion

### 2. Synth-Pop Dreams
**Influences:** Postal Service, MGMT, La Roux
**Core Elements:** Ambient pads, drum machines, synth leads, filtered vocals, arpeggios
**BPM Range:** 80-110 BPM
**Key Sounds Needed:** Ethereal pads, vintage drum machines, bright synth stabs, reverb tails

### 3. Brat Mode
**Influences:** Charli XCX, SOPHIE, 100 gecs
**Core Elements:** Hyperpop, detuned synths, glitch, hard drums, pitched vocals
**BPM Range:** 120-150 BPM
**Key Sounds Needed:** Glitchy FX, detuned hypersaw leads, punchy drums, vocal chops, distorted synths

#### 2. **Naming Conventions**

**Format:** `[Characteristic]_[Detail]_[Source/Version]`

**Examples:**

```
Kick_Deep-808_Splice-v2.wav
Kick_Punchy-909_Loopmasters_Filtered.wav
Snare_Clappy-Vintage_Samples-From-Mars.wav

Vocal-Chop_Pitched-Up_Original.wav
FX_Riser-Synth_Distorted_Custom.wav
Drum-Loop_Breaks-130bpm_Big-Beat-Blowout.wav
```

**Naming Rules:**
- Use underscores instead of spaces
- Use hyphens for multi-word descriptions
- Lead with primary characteristic
- Include source or processing if relevant
- Keep to 50 characters when possible
- Avoid numbers that don't convey meaning (use descriptors instead)

#### 3. **Metadata and Tagging Strategy**

**Essential Metadata Fields:**

```
Instrument: [Drums, Bass, Synths, Vocals, FX, etc.]
Sub-Category: [Kick, Snare, 303, Pad, etc.]
Genre: [Big Beat, Synth-Pop, Hyperpop, Breakbeat, Ambient, etc.]
Mood: [Dark, Bright, Warm, Aggressive, Ethereal, etc.]
BPM: [Tempo if loop/one-shot]
Key: [Musical key if tonal]
Tone: [Warm, Bright, Dark, Dirty, Clean, etc.]
Processing: [Distorted, Filtered, Reversed, Pitched, etc.]
Source: [Splice, Loopmasters, Free, Custom, etc.]
Tags: [Any additional searchable terms]
```

**Recommended Tagging Tools:**

1. **ADSR Sample Manager** (Free VST/AU plugin)
   - URL: https://www.adsrsounds.com/product/software/adsr-sample-manager/
   - Auto-tagging capability
   - Searchable interface within DAW
   - Standalone operation

2. **Ableton Live 12 Browser with Custom Tags**
   - Places system for custom folders
   - New filtering by instrument type, audio format
   - Drag-and-drop organization
   - Real-time search

3. **Serum Wavetable Browser**
   - Custom folder organization
   - Preset tag system
   - Favorites marking

4. **Native Instruments Kontakt Browser**
   - Bookmark system
   - Favorites organization
   - Multiple library locations support

**Tagging Strategy:**
- Keep tags short (1-2 words max)
- Use consistent tag terminology across library
- Create genre-specific tags for your three banks:
  - `big-beat-blowout`
  - `synth-pop-dreams`
  - `brat-mode`
- Add mood/character tags: `punchy`, `ethereal`, `aggressive`, `clean`, `dirty`
- Mark project-specific: `in-use-bbblowout`, `testing`, `archived`

---

#### 4. **Maintenance and Backup Strategy**

**Organization Workflow:**

1. **Quarantine Folder for New Samples**
   - All new downloads go here first
   - Weekly processing cycle
   - Tag, rename, organize, test

2. **Testing Protocol Before Adding to Main Library**
   - Check quality (listen thoroughly)
   - Verify metadata accuracy
   - Test loading in DAW
   - Confirm no corrupted files

3. **No Sample Rule**
   - "No sample enters main library until it's tagged, renamed, and sorted"

4. **Backup Strategy**
   - Main library on fast external SSD (for daily work)
   - Backup copy on second external drive (monthly)
   - Metadata exported separately in CSV format
   - Cloud backup for critical collections (Google Drive, Dropbox)

5. **Quarterly Audit**
   - Review unused samples (over 6 months)
   - Archive or delete duplicates
   - Update outdated metadata
   - Verify all files intact and accessible
