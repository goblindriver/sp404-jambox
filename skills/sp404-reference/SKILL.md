# SP-404A Hardware Reference

## Hardware Overview

The Roland SP-404A (original, not MK2) is a compact phrase sampler with:
- **12 pads** per bank (A through J) = 120 total pad slots
- **Sub Pad**: a hardware retrigger button (rapid-repeat of last-pressed pad for rolls/fills) — NOT a sample slot
- Banks A-B are **internal memory** (survive SD card swaps)
- Banks C-J are stored on the SD card
- Audio: 16-bit / 44.1kHz / Mono / PCM WAV only
- SD card must be **FAT32** formatted
- No project file import — output is WAV files + PAD_INFO.BIN + patterns on SD card

### Key Differences from MK2
- 12 pads per bank (not 16 like MK2)
- No Loop Capture, Groove Function, Sound Generator, or Serato integration
- No firmware-level effects bus routing
- Community resources and paid packs skew MK2 — filter accordingly

## SD Card Structure

```
/Volumes/SP-404SX/
└── ROLAND/
    └── SP-404SX/
        ├── SMPL/           # WAV sample files + PAD_INFO.BIN
        │   ├── PAD_INFO.BIN
        │   ├── A0000001.WAV  ... A0000012.WAV (Bank A)
        │   ├── B0000001.WAV  ... B0000012.WAV (Bank B)
        │   └── ...through J
        └── PTN/            # Pattern files
            ├── PTN00001.BIN ... (Bank A patterns)
            └── ...
```

### File Naming
- `{BANK_LETTER}0000{PAD_NUMBER}.WAV`
- Banks: A through J, Pads: 1-12
- Example: `G0000007.WAV` = Bank G, Pad 7

## PAD_INFO.BIN

Per-pad metadata file at `ROLAND/SP-404SX/SMPL/PAD_INFO.BIN`.
Generated by `scripts/gen_padinfo.py`.

Auto-sets:
- **Gate mode** for pads 1-4 (drum one-shots — sound stops when you release the pad)
- **Loop mode** for pads 5-12 (loops/melodic — sound plays through and loops)

## RLND WAV Chunk

SP-404A WAVs include a proprietary "RLND" chunk (466 bytes) encoding device ID and pad index. Injected by `scripts/wav_utils.py:inject_rlnd()` during sample conversion.

## Pattern Files

Pattern files (.PTN) in `ROLAND/SP-404SX/PTN/` are proprietary binary format. Generated via `scripts/gen_patterns.py` using vendored spEdit404 (`scripts/spedit404/`).

## Effects

The SP-404A has 29 built-in effects. See `references/effects-table.md` for the full list with knob assignments and suggested use cases.

### Key Effects for Jambox

| Effect | Best For | Notes |
|--------|----------|-------|
| Vinyl Sim | Adding warmth/lo-fi character | Classic SP-404 sound |
| DJFX Looper | Glitch, stutter, live chops | Real-time performance tool |
| Isolator | Live mix control, frequency cuts | Essential for transitions |
| Filter + Drive | Aggressive textures, buildups | Combine for intensity |
| Delay | Dub effects, space | Tempo-synced |
| Reverb | Ambient washes, depth | Use sparingly on drums |
| Compressor | Punching up drums, glue | Subtle settings work best |

### Resample Workflow

Resampling is the SP-404's superpower — recording the output back onto a pad:
1. Play a pad (with or without effects)
2. Hold REC + press destination pad
3. The processed audio records to the new pad
4. Layer, stack, and chain for complex textures

Common resample chains:
- Drums → Vinyl Sim → Resample → Compressor → Resample
- Loop → Filter sweep → Resample → Chop sections
- Any pad → Reverb tail → Resample → Use as transition

## Important Limitations

- **44.1kHz/16-bit/mono only** — anything else may not play correctly
- **FAT32 required** — format IN the SP-404A for correct structure
- **No undo** — resampling is destructive (overwrites the destination pad)
- **12 pads per bank** — plan your layout carefully
- **Sub Pad is retrigger only** — do not assign samples to it
- **Always safely eject** the SD card before removing from computer
